# Jovi-v70# 

Este projeto é uma **simulação / protótipo em HTML, CSS e JavaScript** inspirado no *Modo Estudo* do celular **Jovi V70**, com uma interface moderna e animações suaves.

O objetivo é representar um app de produtividade escolar com funções como:
- Ativação de modo estudo
- Controle de internet (apps bloqueados/liberados)
- Câmera IA simulada
- Geração de resumo automático
- Tela de configurações personalizadas

---

## 🚀 Demonstração

📌 Interface inspirada em um smartphone real com:
- Notch
- Barra de status
- Navegação inferior (Bottom Navigation)
- Transições de tela estilo app real

---

## 🧠 Funcionalidades

✅ **Tela Início (Dashboard)**
- Toggle de ativação do Modo Estudo
- Contador de tempo ativo
- Acesso rápido para câmera e internet
- Lista de matérias com progresso

📷 **Câmera IA (Simulada)**
- Visual estilo viewfinder
- Animação de escaneamento
- Overlay de processamento IA

📝 **Resumo IA**
- Mostra preview da foto simulada
- Pontos-chave organizados
- Tags de palavras-chave
- Botões para salvar resumo e criar flashcards

🌐 **Controle de Internet**
- Apps bloqueados e liberados
- Barra de uso de dados
- IA avaliando conteúdo

⚙️ **Configurações**
- Slider de tempo máximo por sessão
- Toggle para notificações e modo noturno
- Configurações da câmera IA e idioma
